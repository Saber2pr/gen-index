# @saber2pr/gen-index

> generate an index.ts file for the dir.

```bash
# from npm
npm install @saber2pr/gen-index

# from github
git clone https://github.com/Saber2pr/gen-index.git
```

# Usage

```bash
genIndex ./src/utils
```

---

## start

```bash
npm install
```

```bash
npm start

```
